// 03 / How it works

From raw chain data to a verdict you can defend.

PreCog turns multi-chain activity into a scored, fully-sourced risk assessment — every figure traceable back to the evidence behind it.

01  / INGEST

Every transaction, every indexed chain.

We index and normalize activity across 6 chains in real time — addresses, transfers, and counterparties resolved into one consistent graph.

02  / SCORE

Named signals, each with its own evidence.

Atlas, Tango and Pedigrid each return a sub-score, a confidence band, and the exact evidence it relied on.

03  / REPORT

A verdict an analyst can defend.

A single weighted risk score arrives with the full reasoning attached — sources, weights, and signal contributions. No black boxes.
